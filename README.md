<h2>一、项目简介</h2>
基于ssh的城市公交查询管理系统：前端jsp、jquery、bjui，后端 struts、spring、hibernate，集成公交查询、线路查询、站点地图、线路地图等功能于一体的系统。

<h2>三、功能介绍</h2>
<div class="markdown-heading" dir="auto">
<div class="markdown-heading" dir="auto">
<h3 dir="auto" tabindex="-1">用户</h3>
<ul dir="auto">
 	<li>基本功能：登录、注册、退出、个人资料查看、个人资料修改</li>
 	<li>公交查询：从发点-终点查询、线路查询结果</li>
 	<li>线路查询：车次名称查询、站点名称查询、车次线路列表</li>
 	<li>站点地图：从发点-终点查询、地图上展示结果、地图上路径规划的曲线</li>
 	<li>线路地图：输入线路查询、地图显示线路和站点信息</li>
 	<li>投诉建议：投诉信息提交</li>
</ul>
<h3 dir="auto" tabindex="-1"><a id="user-content-管理员" class="anchor" tabindex="-1" href="https://github.com/YuLin-Coder/No47UrbanPublicTransportQueryManagementSystem#%E7%AE%A1%E7%90%86%E5%91%98" aria-hidden="true"></a>管理员</h3>
<ul dir="auto">
 	<li>站点管理：站点列表、站点名称搜索、站点信息修改、站点信息删除、添加站点、批量删除</li>
 	<li>线路管理：车次搜索记录、线路列表、线路信息修改、线路删除、添加线路、批量删除、距离配置</li>
 	<li>车辆管理：车辆列表、车牌号搜索车辆信息、添加车辆、删除、批量删除</li>
